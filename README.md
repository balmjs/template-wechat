# BalmJS template - wechat-mp

> A simple project for WeChat Media Platform

## Installation

```sh
npm i -g balm-cli
balm init wechat-mp my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```

> thx [BalmJS](https://balmjs.com/)
